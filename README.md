# python-exam

## 文档

### 需要掌握的知识

* Python 3.11+（解释器性能提升23%）
* 类型提示系统（Type Hints）
* 异步编程基础（async/await语法）


### 特性

* Python 依赖缩进（4个空格）表示代码块，不要混用空格和 Tab
* 动态类型语言，注意变量类型变化
